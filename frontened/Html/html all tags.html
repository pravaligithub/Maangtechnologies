A logic gate is a device that acts as a building block for digital circuits.They perform basic logical functions that are fundamental to digital circuits.

Most electronic devices we use today will have some form of logic gates in them. Logic gates are used to make decisions so that electrical outputs only 'turn on' when the correct logic sequence has been applied.

Each logic gate has a name that helps to describe how different inputs will determine the possible outputs. The basic logic gates are categorized into seven types as AND, OR, XOR, NAND, NOR, XNOR, and NOT. These are the important digital devices, mainly based on the Boolean function.Logic gates are used to carry out the logical operations on single or multiple binary inputs and result in one binary output.
